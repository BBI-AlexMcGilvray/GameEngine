- to generate cmake files, this is a helper powershell command: Get-ChildItem -Include *.h -Recurse | Format-List FullName | Out-File .\Temp.txt
- 'Materials' contains the core elements used in all parts of 'The Plant'
- 'Pipeline' dictates how different pieces fit together
	- Internals of the game engine
- 'Factory' forms the connections for the 'Pipeline'
	- Editor in to modify/create data
- 'Materials' are passed in and out of the 'Factory'
	- The assets used to generate the assets used in the game, and the assets used in the game
- 'Supplies' are the specific-use materials generated by the 'Factory'
	- The generated code that is then used by the game
- 'Product' is the result of everything
	- Actual game that can be run
